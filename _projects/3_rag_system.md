Developed a production-ready Retrieval-Augmented Generation (RAG) architecture tailored for highly contextual semantic question-answering over unstructured PDF documentation.

### Key Implementation Details

- **Pipeline Engineering:** Built robust document parsing, text chunking strategies, and vector embedding generation pipelines utilizing **LangChain** and **ChromaDB**.
- **Inference & UI:** Exposed underlying LLM orchestration logic through highly optimized **FastAPI** REST endpoints and built a seamless, interactive chat interface using **Streamlit** for real-time document ingestion and conversational querying.
