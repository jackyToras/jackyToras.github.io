Built a high-throughput, distributed message queue designed to handle intensive asynchronous background tasks.

- **Workload Balancing:** Implemented load-balancing algorithms to dynamically distribute workloads across multiple active worker nodes, ensuring system stability under heavy traffic spikes.
- **Decoupling Architecture:** Orchestrated using Java, Spring Boot, and RabbitMQ to manage task lifecycles with fault tolerance and strict message delivery guarantees.
